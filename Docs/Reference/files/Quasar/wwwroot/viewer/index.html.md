# Quasar/wwwroot/viewer/index.html

Static shell for the Quasar grid viewer. It loads the viewer CSS and ES modules, provides controls for scene reload, local Space Engineers `Content` and Mods folder selection, hidden folder file input fallbacks for browsers without `showDirectoryPicker`, render toggles including `Show lighting` and `Show logistics`, stats, warning logs, hover readout, and the full-window Three.js viewport.
