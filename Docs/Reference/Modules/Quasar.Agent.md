# Quasar.Agent — In-DS Plugin

*Module `Quasar.Agent` — 12 files.* See the [handbook TOC](../TOC.md) and the [file Index](../Index.md).

The plugin loaded inside each Space Engineers Dedicated Server (`netstandard2.0`, referencing the VRage/Sandbox game assemblies and the Magnetar PluginSdk provided at runtime). `AdminPlugin` is the `IPlugin` entry point; `AgentConnection` runs the WebSocket reconnect loop, sending hello/snapshot/plugin-config messages and executing inbound commands; `GameBridge` and `EntityInspector` marshal telemetry collection and command actions onto the game thread; and `WebServiceLocator` finds an existing supervisor or triggers [Quasar.Bootstrap](Quasar.Bootstrap.md). If the supervisor stays offline beyond a configurable window, the agent autonomously saves and stops the server. It speaks the [Magnetar.Protocol](Magnetar.Protocol.md) contracts.
