# Quasar.Agent/GridRenderSceneInspector.cs

**Module:** Quasar.Agent  **Kind:** class  **Tier:** 1

## Summary
Background scene-capture helper that builds viewer scene snapshots from live `MyCubeGrid` and asteroid `MyVoxelBase` instances without marshalling the expensive DTO collection through the game thread. It maps primary and optional context grids into `Magnetar.Protocol` viewer DTOs, computes selected-grid-local context bounds plus matching relative/world AABBs and an internal snapped 3D clip prism derived the same way as the browser floor-grid clip, enumerates loaded grids deterministically, admits ordinary context grids through the clip-prism world AABB plus block-level prism intersection instead of recursively expanding context-grid mechanical links, sends only prism-intersecting blocks for non-primary context grids, keeps the selected primary grid full-rendered, selects voxel bodies and voxel samples against the same snapped X/Y/Z prism used by browser clipping, registers active scene mod asset roots and logical model/texture names with mod root hints, includes generated cube-part and runtime subpart metadata with subpart entity IDs, conservatively culls generated single-cell armor faces hidden by adjacent undamaged same-grid generated tiles marked `FullQuad` by Space Engineers topology metadata and omits fully culled generated blocks from the scene, exports cached emissive material states including light-specific render-property bulb updates, captures current-model orientation matrices plus build level, integrity, max integrity, accumulated damage, and damaged-block skeleton offsets, captures scene-wide conveyor/logistics topology and conveyor reachability so online links across visible grids share highlight systems without marking context-clipped but still connected endpoints as dangling, captures terminal-synced lighting and LCD metadata, assigns grid ownership on blocks/chunks/lights, and emits separate voxel damage chunks with per-sample masks produced by comparing current storage against the voxel data provider's original terrain.
